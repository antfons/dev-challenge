#### Descrição
Projeto para demonstrar um básico aprendizado de angular, utilização de componentes, angular material e fundamentos como interpolação.

## Dependências

```console
npm install
```

### Executando a aplicação
```console
ng serve/ng serve -o
```

### Abrindo a aplicação
```console
http://localhost:4200
```
